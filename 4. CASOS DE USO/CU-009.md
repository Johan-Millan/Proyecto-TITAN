CU-009 – Generar Informe Diario de Cursos Ejecutados
Información General

ID: CU-009

Nombre: Generar Informe Diario de Cursos Ejecutados

Descripción breve: Permite al personal administrativo y gerencia generar automáticamente un informe diario con los cursos realizados, incluyendo participantes e instructores.

Módulo: Reportes y Cumplimiento

Prioridad: MUST

Complejidad: Media

Estado: En Revisión

Actores

Actor Principal: Personal administrativo / Gerencia

Actores Secundarios: Sistema

Precondiciones

El sistema debe tener los registros actualizados de cursos y participantes. (RF-002.2)

El usuario debe estar autenticado con permisos adecuados. (RNF-003.1)

Postcondiciones

El informe diario queda generado y disponible para descarga o consulta. (RF-002.2)

La acción queda registrada en el historial de auditoría. (RNF-003.4)

Flujo Principal (Camino Feliz)

El usuario accede al módulo “Informes diarios”.

Selecciona la fecha del informe.

El sistema recopila automáticamente los datos de cursos ejecutados, participantes e instructores.

El sistema genera el informe en formato PDF o Excel.

El sistema muestra el informe al usuario y permite descarga o envío por correo.

La acción queda registrada en auditoría.

Flujos Alternativos

A1 – Enviar informe por correo:

El usuario puede solicitar que el informe se envíe automáticamente a un correo corporativo.

El sistema envía el informe al destinatario indicado. (RF-002.2)

Flujos de Excepción

E1 – Error en la recopilación de datos:

El sistema notifica error y solicita reintentar la operación.

Se registra el error en auditoría. (RNF-003.4)

E2 – Usuario no autorizado:

El sistema bloquea la generación del informe y muestra mensaje de acceso denegado. (RNF-003.1)

Reglas de Negocio

RN-16: Solo usuarios con permisos administrativos o de gerencia pueden generar informes diarios. (RF-002.2)

RN-17: Todos los informes generados deben registrarse en el historial de auditoría. (RNF-003.4)

Requisitos No Funcionales

RNF-001.1: Interfaz simple y clara para generación del informe.

RNF-002.2: Generación de informes en tiempo de respuesta aceptable.

RNF-003.3: Protección de datos de participantes e instructores.

RNF-003.4: Registro de la operación en auditoría.

Interfaces de Usuario

Pantalla de selección de fecha y filtros de cursos.

Vista previa del informe generado.

Botones “Descargar” y “Enviar por correo”.

Referencias

RF-002.2: Generación de informes diarios de cursos ejecutados.

RNF-003.1: Autenticación de usuarios.

RNF-003.3: Protección de datos sensibles.

RNF-003.4: Registro de auditoría.
